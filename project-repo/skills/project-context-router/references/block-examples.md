# Block Examples

## Input
"NPO: клиент согласовал концепт. Нужно до 7 марта обновить копирайт. Риск: дизайнер может не успеть с баннерами"

## Output

```markdown
## NPO

### Статус
- Клиент согласовал концепт.

### Задачи
- [ ] Обновить копирайт.

### Дедлайны
- До 7 марта обновить копирайт.

### Риски
- Дизайнер может не успеть с баннерами.
```
