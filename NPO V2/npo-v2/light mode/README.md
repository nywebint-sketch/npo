# NPO v2

Рефакторнутый статический прототип без визуального редизайна.

## Структура

- `index.html` — разметка страниц и секций.
- `styles.css` — стили и UI-состояния.
- `js/main.js` — точка входа `initApp()`.
- `js/data.js` — данные карточек и дефолтные конфиги.
- `js/dom.js` — кэш DOM-узлов.
- `js/render.js` — рендер сеток и контента модалок.
- `js/ui.js` — меню, модалка, навигация, toast-уведомления.
- `js/booking.js` — логика отправки букинг-форм.
- `js/auth.js` — логика авторизации клуба.

## Конфигурация endpoint'ов

Через `<meta>` в `index.html`:

- `booking-endpoint`
- `club-api-base`

Если `booking-endpoint` не задан, используется дефолтный тестовый endpoint из `js/data.js`.
